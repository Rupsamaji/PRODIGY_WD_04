# Personal Portfolio Website

This project is a personal portfolio website created to showcase my technical skills, projects, and background as a Computer Science student. The website provides an overview of my work, interests, and contact information in a clean and interactive design.

## Features

- Modern and responsive design
- Interactive navigation with smooth scrolling
- Project showcase section
- Skills section highlighting technical abilities
- About section describing my background and interests
- Contact section for easy communication
- Hover animations and interactive UI elements

## Technologies Used

- HTML
- CSS
- JavaScript

## Sections Included

- Home
- About
- Skills
- Projects
- Contact

## Purpose

The goal of this portfolio is to create a professional online presence where recruiters, collaborators, and developers can explore my projects and technical skills.
